Overview
========
This directory stores a HSE theme (name="hse2") for an Open edX instance.

Theme Authoring
===============
- Edit the lms.envs.json file in edx-platform and set 'USE_CUSTOM_THEME' to true, and 'THEME_NAME' to your theme's name.
